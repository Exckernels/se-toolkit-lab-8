# LMS Skill

This skill defines how the agent should use LMS MCP tools.

Use LMS tools whenever the user asks about:
- labs
- pass rates
- scores
- learners
- interactions
- LMS analytics
- backend data

Guidelines:
- Always prefer LMS tools over guessing.
- If the question requires data, call the appropriate LMS tool.
- If a lab is not specified, ask a clarification question.
- Use exact lab names returned by tools.
- Do not hallucinate data.
- If no data is returned, clearly say that no data is available.
- Keep answers concise and factual.
- For architecture questions, combine general knowledge with LMS data when relevant.
